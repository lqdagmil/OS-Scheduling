# OS-Scheduling
OS Scheduling Technique in C#

Supported Technique
- FCFS
- SJF (Non Pre-emptive)
- SJF (Pre-emptive)
- Priority (Non Pre-emptive)
- Priority (Pre-emptive)
- Round-Robin

Limitation
- Upto 26 Processes (can be change in the source code)
- Limited Gantt chart view
- Quantum locked at (1 - 9)
